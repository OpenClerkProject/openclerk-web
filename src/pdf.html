This page is a separate bundle from the rest of OpenClerk because it loads pdf.js and tesseract.js -- together several MB -- so visitors to the lightweight Citation Checker and Document Editor pages never pay that cost. The PDF you upload is parsed entirely in your browser and never leaves it; OCR (only run on pages with no embedded text) is fully self-hosted too -- its worker, WASM engine, and language model are all served from this same site, so nothing is fetched from a third party.

Free token from courtlistener.com/profile/api-token. Only the matched citation strings are sent, never the document text.
